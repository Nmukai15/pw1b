Tags aprendidas

Tag img auto fechante

A tag img é usada para inserir uma imagem na página
  • Requer o atributo src para definir o caminho da imagem

Tag div

Usada para agurpar e organizar o conteúdo em blocas na página

Este é um parágrafo dentro da div

Tag input autofechante

A tag input permite que o usuário isira dados no site.

Pode ser configurada para difentes tipos de entrada como:
  1. Texto;
  2. Senha;
  3. Data;
  4. Número, etc.

Tag button

A tag button é usada para criar um botão interativo que pode ser clicado pelo usuário.

Tag A

tag a no HTML é usada para criar links, permitindo que os usuários naveguem para outras páginas ou recursos. Ela tem o atributo href, que define o URL de destino.
